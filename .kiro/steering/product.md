# Product Overview

**DOWN** is a 2D vertical platformer browser game where players race to reach the lowest point of each level as fast as possible. The game features timer-based scoring with a competitive leaderboard system.

## Core Concept
- Vertical descent gameplay (going DOWN)
- Timer-based scoring system
- Avoid obstacles and enemies while collecting time crystals
- Submit scores to a global leaderboard

## Key Features
- Tutorial system teaching movement, obstacles, and enemies
- Health system (3 hit points)
- Time crystals that pause the timer
- Environmental hazards (spikes, slow surfaces)
- Ground and flying enemies
- PostgreSQL-backed leaderboard

## Target Platform
Browser-based game playable on desktop devices with keyboard controls.

## Development Status
Early development phase with core mechanics and tutorial flow implemented. Obstacle entities, enemies, and full scoring system are in progress.
