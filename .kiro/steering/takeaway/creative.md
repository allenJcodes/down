# Creative Learnings

Track creative insights and design discoveries throughout the DOWN platformer project development.

---

## Game Design

### Learnings
- Tutorial progression (movement → obstacles → enemies) feels natural and intuitive
- Exit zones work better than edge triggers for giving players control
- Invulnerability frames are essential for fair damage mechanics
- Clear visual indicators (green exit zones) help guide player behavior

## User Experience

### Learnings
- Visual feedback (color flashes) is crucial for player understanding
- Timer display needs to be scroll-independent in vertical levels
- Explicit trigger zones provide better player control than automatic transitions

---

## Visual Design

### Learnings
- Placeholder graphics (colored rectangles) allow rapid prototyping without blocking progress
- Color coding helps communicate mechanics (red = danger, green = safe/goal, blue = player)
